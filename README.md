# MyPlanGallery

A responsive, premium-grade web application for comparing Medicare Advantage plans.

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Use the provided `deploy` script to publish to GitHub Pages:

```bash
npm run build
npm run deploy
```

Ensure the `base` in `vite.config.ts` matches your repository name.
This build uses a teal accent color (#009FDA).
